# Temporal Bundle

## Requirements:
- php >= 8.2
- symfony >= 6.3


## Installation:

1. Connect recipes

```bash
composer config --json extra.symfony.endpoint '["https://github.com/VantaFinance/temporal-bundle/-/raw/master/.recipie/index.json", "flex://defaults"]' 
```



2. Install package

```bash
composer req vanta/temporal-bundle    
```
